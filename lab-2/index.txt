<h1> Javascripts </h1> <!-- 8 --> 
<h1> Introduction </h1>
<p> JavaScript was initially created to “make web pages alive”. The programs in this language are called scripts. They can be written right in a web page’s HTML and executed automatically as the page loads. Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run. In this aspect, JavaScript is very different from another language called Java. 
</p>

<h2> Features </h2>
<ul>
  <li>1. Universal support</li>
  <li>2. Imprative and structured</li>
  <li>3. Dynamic typing</li>
  <li>4. Object-oriented or prototype-based</li>
</ul>
