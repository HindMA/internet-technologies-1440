Javascripts

Introduction

JavaScript was initially created to “make web pages alive”. The programs in this language are called scripts. They can be written right in a web page’s HTML and executed automatically as the page loads. Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run. In this aspect, JavaScript is very different from another language called Java.

Features

  1. Universal support
  2. Imprative and structured
  3. Dynamic typing
  4. Object-oriented or prototype-based
